# Emoji Mouse Trace

Trace mouse movement with a trail of emojis or images.

## API
| Parameter        | value | default |
| ------------- |:-------------:|:-------------:|
| traceLength      | int | 20 |
| imageSize      | int | 30  |
| images | url | none |
| fadeOut | boolean | false |
| mode | emoji or images | emoji |
| fontFamily | - | Arial |
| fontSize | px | 30px |
| emojis | string | random emojis  |
